Примеры совсем простых вопросов

Q: Что такое переменные? зачем они нужны? Что такое тип переменной?
A: Переменные - это именнованные участки памяти, предназначенные для хранения каких-либо данных внутри программы, к которым можно обращаться для чтения и записи. Тип переменной определяет её размер и формат данных, которые в ней хранятся (например целые числа, строки, символы, вещественные числа)

Q: Как объявить переменную?
A: тип_перемнной имя_переменной;
int a;

Q: Как считывать данные с клавиатуры и выводить их на экран?
A: Используя cin, cout

Q: Напишите небольшой фрагмент кода для следующей задачи: На вход программе подаётся 10 чисел, необходимо вывести их сумму.
A: 

                    int sum = 0;
                    int num;
                    for (int i = 0; i < 10; ++i) {
                        cin >> num;
                        sum = sum + num;
                    }
                    cout << sum << endl;

Q: Как переписать цикл for в цикл while и обратно
A: 

                    for (I; II; III) {
                        IV;
                    }

                    //
                    I;
                    while (II) {
                        IV;
                        III;
                    }

Q: Что такое массивы?
A: Массивы - это специальные переменные, предназначенные для хранения сразу нескольких значений одного типа. Для объявления массива необходимо указать его размер.

Q: Как обращаться к элементам массива?
A: Используя []. Внутри квадратных скобок следует указать индекс значения, к которому хочется обратиться для чтения или записи.

Q: Зачем нужно динамическое выделение памяти?
A: Динамическое выделение памяти позволяет выделять дополнительную память прямо во время выполнения программы.

Q: Что такое объявление переменной? Что такое инициализация переменной?
A: Объявление переменной - инструкция компилятору, которая включает в себя тип и имя переменной. Объявление переменных необходимо для их дальнейшего использования. Инициализация переменной - это задание начального значения для переменной.
